# Setup

Download: 
- [NPM](https://www.npmjs.com/get-npm)
- [Atom](https://atom.io/)

Run the following commands:
```
npm install -g truffle
npm install -g ethereumjs-testrpc
```

To compile the project run:
```
truffle compile
```

To migrate the project run:
```
truffle migrate
```

To test the project run:
```
truffle test
```
